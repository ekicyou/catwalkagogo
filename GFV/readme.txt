GFV - Graphic File Viewer

*概要
WPFとXnViewのGFL(Graphic File Library)を使った画像ビューア。シングルインスタンス・マルチウインドウによるSDI(シングルドキュメントインターフェイス)。

.NetFramework4.0、WindowsXP以降。

*特徴

**多彩な対応画像フォーマット
GFLにより、100を越える多彩な画像フォーマットに対応します。

ver1.0.2.0からWIC(Windows Imaging Component)に対応し、カメラメーカーなどが提供するコーデックをインストールすることで対応画像形式を増やせるようになりました。

**表示領域を出来るだけ大きく
ウインドウの枠を取り払うなどして出来るだけ画像表示領域を広くしています。

**シングルインスタンスSDI
SDIインターフェイスを採用しています。WindowsXP以降のタスクのグループ化や、Windows7のタスクスイッチを利用します。

シングルインスタンスなのでウインドウを多数開いてもメモリ使用量を抑えられます。

マルチモニターに対応したウインドウの並び替えが可能です。
