GFV - Graphic File Viewer

*概要
WPFとXnViewのGFL(Graphic File Library)を使った画像ビューア。シングルインスタンス・マルチウインドウによるSDI(シングルドキュメントインターフェイス)。

.NetFramework4.0、Windows Vista以降向け。

*特徴

**多彩な対応画像フォーマット
GFLにより、100を越える多彩な画像フォーマットに対応します。

**表示領域を出来るだけ大きく
ウインドウの枠を取り払うなどして出来るだけ画像表示領域を広くしています。

**GPUによる支援
画像の拡大縮小にはWPFを使っているのでGPUによる支援を受けられます。

※ただし、GFLをラップしている都合上、オーバーヘッドは存在します(ロード時)。
